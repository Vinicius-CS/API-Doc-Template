The example API uses the following error codes:

Error Code Meaning
X000 Some parameters are missing. This error appears when you don't pass every mandatory parameters.
X001 Unknown or unvalid Authorization. This error appears if you use an unknow token or if your token expired.
X002 Unvalid Authorization for this domain. This error appears if you use an token non specified for your domain. Developper or Universal token doesn't have domain checker.
X003 Unknown or unvalid user token. This error appears if you use an unknow user token or if the user token expired.
